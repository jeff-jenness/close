# Literals #

## Integer Constants ##
> - various 0x... formats
> - u/l/ll suffixes
## Floating-Point Constants ##
> - decimal
> - hex
> - f/l suffixes
## Character Constants ##
> - there are no character constants, only very short strings.
## String Literals ##
> - single quoted
> - double quoted
> - heredocs
## Aggregate Literals ##
> - syntax for structures
> - named element initialization
> - etc.
